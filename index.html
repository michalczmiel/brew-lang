Brew Language

A DSL for writing coffee brew recipes. Create and share pour-over recipes with precise timing and temperature control.

Basic Syntax

  • dose 15 — Coffee weight in grams
  • temperature 90 — Water temperature in °C
  • brewer origami — Brewing device
  • filter paper — Filter type
  • -- comment — Comments

Timed Steps

at 0:00..0:10
  pour 50
end
at 2:00
  swirl
  -- gently
end

Editor Features

  • Timer — Interactive brew timer
  • Diagram — Visual timeline
  • Linting — Real-time syntax validation
  • Status bar — Auto-calculates ratio & total water
  • Share — Copy recipe URL
  • Vim mode — Available in settings

This project is in active development.
